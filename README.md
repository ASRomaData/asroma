# 🔴 AS Roma Stats Pro

App web + automazioni per tracciare e pubblicare le statistiche della Roma.

## 🚀 Deploy
1. Carica su GitHub
2. Importa su Vercel
3. Fine

## 🤖 Automazioni
Incluso GitHub Actions per chiamare API ogni giorno.

### 🔐 Secrets GitHub richiesti
Vai su:
Settings → Secrets → Actions

Aggiungi:
- BSKY_HANDLE
- BSKY_PASSWORD

## 📅 Workflow
- Esegue ogni giorno
- Può essere esteso per post automatici

## 📁 Struttura
- /api → backend
- /public → frontend
- /.github/workflows → automazioni
